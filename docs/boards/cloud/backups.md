# Boards Cloud Backup Policy

Regular database and object storage backups are taken for disaster recovery and business continuity purposes. 
These backups are retained for a period of 4 weeks. After which they are deleted.

| Frequency Unit  | Every    | Retention Time | Snapshot Time |
| --------------- | -------- | -------------- | ------------- |
| Hourly Snapshot | 2 hours  | 3 days         | 02:28 UTC     |
| Daily Snapshot  | 1 day    | 7 days         | 02:28 UTC     |
| Weekly Snapshot | Saturday | 4 weeks        | 02:28 UTC     |
