# Keyboard Shortcuts

|                 | Shortcut        | Notes                                                                                             |
| --------------- | --------------  | ------------------------------------------------------------------------------------------------- |
| Copy            | Ctrl+C          | Copy the Card/List that your mouse is highlighting                                                |
| Cut             | Ctrl+X          | Cut the Card/List that your mouse is highlighting                                                 |
| Paste           | Ctrl+V          | Paste a Card/List. Make sure your mouse is highlighting a List or Card as the target              |
| Toggle Complete | Space Bar       | Highlight a Task with your mouse to toggle on/off the complete status                             |
| Delete          | Delete          | Hover the card and press Delete to archive the card                                               |
| Create ToDo     | Enter           | Once you've typed your card name, create a Task (tick box)                                        |
| Create Note     | Shift+Enter     | Once you've typed your card name, create a Note                                                   |
| Close Card      | ESC             | Close the currently open Card Dialog                                                              |
